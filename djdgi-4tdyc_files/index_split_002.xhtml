二  Calibre－電子書管理軟體

Calibre 是一款功能齊全，自由開源的電子書管理軟體。

除了基本的電子書書庫管理、元數據編輯／下載、內建電子書閱讀器等用途外。Calibre 還內置了十種以上的電子書格式互轉、與手持閱讀設備進行互動、將新聞網站的 RSS 供稿製成電子書、個人書庫遠端分享等功能。運行方面，Calibre 支援Mac、Linux和 Windows 三種作業系統。 

 (** 但不支援Android 手機 **) 

Windows 或 OS X 使用者，前往官網下載、安裝： 

https://calibre-ebook.com/download  

安裝完成，出現The  calibre main program軟體圖示 (可釘選至「工作列」)

Calibre內含：

1  The calibre e-book editor 

2  The calibre e-book viewer 

3  The calibre main program 

在〈轉換書本〉、〈路徑：點擊以開啟〉、 開啟電子書文件夾(*.epub)、選第二項〈開啟檔案〉時可以看到上述三種軟體和圖示
